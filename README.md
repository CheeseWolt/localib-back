# Localib-back

## Description

Ce repo contient la partie métier de l'app [Localib](https://github.com/CheeseWolt/Localib). C'est une API REST qui permet de gérer des locations de véhicules.



## Installation

```bash
$ npm install
```

## Lancer le back

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
